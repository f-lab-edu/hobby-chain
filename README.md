# hobby-chain
### 💬 취미 공유 SNS 서비스 'hobby-chain' 개발 프로젝트
- 프론트 엔드는 생략하고 백엔드에 초점을 맞춰 REST API를 개발했습니다.


## ⌨️ 사용 기술 및 개발 환경
**Language** : Java

**Framework** : Springboot

**Database / ORM** : MySQL, MyBatis

**Session Storage** : Redis

**IDE** : IntelliJ

**Development tools**: Gradle, Github, Slack

6. 좋아요 기능
