# hobby-chain
### 💬 취미 공유 SNS 서비스 'hobby-chain' 개발 프로젝트
- 프론트 엔드는 생략하고 백엔드에 초점을 맞춰 개발


### ✏️ 사용 기술 및 개발 환경
Java
Spring Boot
MySQL
Redis
Gradle
mybatis
IntelliJ


### 🔎 주요 기능
1. 회원가입 / 탈퇴
2. 로그인 / 로그아웃
3. 회원정보 수정
4. 게시글 작성 / 수정 / 삭제 / 조회
5. 피드 조회
6. 좋아요 기능
7. 댓글 작성 / 수정 / 삭제 / 조회
8. 팔로우 / 언팔로우
